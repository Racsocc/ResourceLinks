# ResourceLinks Typecho 插件

ResourceLinks 是一款功能强大的 Typecho 资源链接管理插件。它允许你在文章或页面底部优雅地展示各种资源链接（如官网、网盘、代码仓库等），并提供丰富的自定义选项和交互功能。

## ✨ 主要特性

*   **多种资源类型支持**：
    *   🔗 **普通链接**：适用于官网、演示地址等。
    *   💾 **网盘链接**：支持填写提取码，前台自动显示复制按钮。
    *   📦 **代码仓库**：适用于 GitHub、Gitee 等项目地址。
*   **优雅的展示风格**：
    *   支持 **列表模式** 和 **卡片模式** 两种展示风格。
    *   内置精美 SVG 图标，自动识别资源类型。
    *   完全响应式设计，完美适配移动端。
*   **可见性控制**：
    *   支持 **回复可见**：隐藏资源链接，要求用户评论后才能查看（有效提升互动率）。
    *   支持全局默认设置和单篇文章独立设置。
*   **便捷的管理交互**：
    *   集成在文章编辑页面底部，无需跳转。
    *   支持拖拽排序。
    *   支持一键复制网盘提取码。

## 🛠️ 安装方法

1.  下载本插件，解压得到 `ResourceLinks` 文件夹。
2.  将 `ResourceLinks` 文件夹上传至 Typecho 的 `usr/plugins/` 目录下。
3.  进入 Typecho 后台，在 **控制台** -> **插件** 中找到 **ResourceLinks**，点击 **启用**。

## ⚙️ 配置说明

启用插件后，你可以点击 **设置** 进行全局配置：

*   **区块标题**：自定义资源列表上方的标题（默认为“资源”）。
*   **显示风格**：选择“列表”或“卡片”布局。
*   **显示图标**：是否显示资源类型的图标。
*   **默认可见性**：设置默认是“公开”还是“回复后可见”。

## 📖 使用指南

1.  撰写新文章或编辑旧文章。
2.  在编辑器下方找到 **资源链接** 面板。
3.  填写资源信息（类型、标题、链接、提取码等），点击 **保存资源**。
4.  你可以添加多个资源，并通过拖拽调整顺序。
5.  如果需要特殊的可见性（如某篇文章想强制回复可见），可在面板右侧调整 **资源可见性**。
